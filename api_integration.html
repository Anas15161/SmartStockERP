Documentation API REST

Guide d'intégration et spécifications des points de terminaison SmartStock ERP.

BASE_URL: http://localhost:8080/smartstock-erp/api

Introduction

L'API de SmartStock ERP est construite avec Jakarta RESTful Web Services (JAX-RS). Elle permet une communication fluide entre le backend Java EE et des clients tiers (applications mobiles, tableaux de bord externes, scripts d'automatisation).

Points de Terminaison (Endpoints)

GET /products

Récupère la liste complète de tous les produits enregistrés en base de données.

Réponse (JSON) :

[
  {
    "id": 1,
    "name": "Ordinateur Portable Dell",
    "quantity": 25,
    "price": 1200.50,
    "sku": "DELL-XPS-13"
  },
  ...
]
GET /products/stock

Récupère l'état actuel des stocks. Identique à la liste des produits mais optimisé pour l'affichage des niveaux.

GET /hello

Service de test pour vérifier la disponibilité de l'API.

Réponse (Text) :

SmartStock ERP: API is running!

Intégration Frontend

Voici un exemple d'utilisation de l'API avec l'API standard JavaScript fetch :

async function fetchProducts() {
    try {
        const response = await fetch('http://localhost:8080/smartstock-erp/api/products');
        if (!response.ok) throw new Error('Erreur réseau');
        
        const products = await response.json();
        console.log('Produits chargés :', products);
        
        // Exemple d'affichage dans le DOM
        products.forEach(p => {
            console.log(`Produit: ${p.name} | Stock: ${p.quantity}`);
        });
    } catch (error) {
        console.error('Erreur lors de la récupération :', error);
    }
}

Sécurité & CORS

Avertissement Actuellement, l'API est accessible au sein du même domaine. Pour des appels cross-domain (CORS), un ContainerResponseFilter doit être implémenté pour autoriser les origines externes.
